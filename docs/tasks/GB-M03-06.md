## Non-negotiable invariants

- Authoritative health zero in a permadeath-enabled instance chooses exactly one terminal outcome. A committed death wins same-tick Recall/extraction conflicts; a server crash is never itself a legitimate death.
- Every normal committed death creates exactly one memorial at every level. Practice/tutorial/developer deaths create no durable death, destruction, memorial, or Echo records.
- All `AtRiskEquipped` and `AtRiskPending` custody is destroyed in canonical order; CharacterSafe, Vault, account currency, cosmetics, codex, recipes, mastery, and settings are preserved.
- The stored result, not mutable live state, is the authority for retry, summary, memorial, telemetry, and support consumers.
- A qualifying death writes death, destruction, memorial, and Echo state in one SQL transaction or writes none. No asynchronous projector may repair a committed qualifying death.
- Later character, item, Bargain, progression, or world mutations cannot commit after final death.
