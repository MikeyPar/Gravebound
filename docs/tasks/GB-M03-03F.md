## Acceptance criteria

1. Every state/event combination maps to exactly one legal transition; duplicates, stale events, impossible terminal reversals, missing content, and mismatched destination snapshots fail closed under deterministic tests.
2. `ready` requires both the exact authoritative location/version and matching compiled scene. Recoverable failure retains the prior safe state; fatal failure offers no unsafe retry; no client state predicts death, extraction, inventory, or reward terminality.
3. LinkLost begins immediately, resolves only from the server-owned 90-tick result, rebinds the same state before terminality, and routes committed extraction/Recall, death, duplicate-session, and crash outcomes exactly once.
4. Strict localization closure proves complete en-US keys for every phase, action, error, and handshake rejection; runtime transition copy cannot bypass the compiled package.
5. The real-QUIC baseline traverses the complete disposable fixed route and arrives at `HallDefault` after the committed Caldus receipt without direct coordinator calls for route transitions or any inventory mutation.
6. The full adverse matrix preserves idempotency, committed-terminal precedence, checkpoint/restore ordering, and safe-state recovery across response loss, disconnect, reconnect, duplicate session, content mismatch, allocation failure, and restart.
7. Normal Character Select `Play`, production Realm Gate admission, normal extraction/Recall, seeded branch traversal, affected Hall stations, production namespaces, and Core promotion remain unavailable and have explicit negative tests.
8. Inspected optimized evidence at both required resolutions, reduced effects, and ultrawide preserves readable state/action hierarchy, keyboard focus, long-copy wrapping, non-color meaning, and the protected combat corridor.
9. Measured performance satisfies the applicable `TECH-070` frame/memory thresholds on the recorded machine, login-to-control median is below 30 seconds, the 30-minute soak has no monotonic leak, and results retain exact build/content hashes.
10. Focused/workspace tests, warnings-denied Clippy, strict content validation, deterministic traces, generated schemas, optimized Windows construction, mandatory PostgreSQL transactions, and hosted CI pass before the completion audit.
