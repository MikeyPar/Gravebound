# GB-M01-08A - Authoritative debug overlay

## Task contract

**Feature IDs:** `GB-M01-08A`, `PRD-123`, `COM-009`, `SIM-011`, `CONT-FP-001`, `CONT-FP-009`
**Depends on:** simulation/content state owners and integrated encounter flow
**Persistence:** developer-only presentation; no gameplay mutation.

## Required behavior

- Toggle with F3 without changing authoritative simulation state or its hash.
- Show selected seed in hexadecimal; default `B311A501`.
- Show run/encounter/combat tick and exact wave/reward/boss state.
- Show arena anchors/spawns, player/enemy hitboxes, pattern timeline, current threat, and active hazards.
- Show a stable gameplay-only state hash with sorted entity ownership.
- Show entity/enemy/friendly/hostile projectile counts and rolling FPS/p95/p99 frame time.
- Mark every surface `DEBUG ONLY`; exclude UI visibility/frame timing from gameplay hash.
- Fail closed on nonfinite player state, invalid health, count overflow, or ambiguous enemy ordering.

## Acceptance criteria

1. Snapshot hash replays identically and changes for gameplay mutation.
2. F3 changes presentation only.
3. Overlay state agrees with the actual integrated Wave 1–3/Boss encounter owner.
4. Seed fixture reproduces exact spawn/attack/damage/drop ticks and hashes.
5. Optimized complete frame is readable and cumulative local gates pass.

## Current implementation state

- Stable gameplay-only hash (including enemy positions), frame/entity counters, F3 toggle, seed, arena/hitbox/pattern/threat fields, and semantic evidence gating are implemented.
- ADR-015 connects ordinary/debug LocalLab to the authoritative encounter director and a duplicate-safe generalized normal-wave owner. Exact Wave 1–3 rosters, global tick offsets, one 27-tick spawn warning, real health deaths, cleanup, drops, reward transitions, and run-qualified identities are no longer simulated by the fixed showcase trio.
- The optimized debug overview visibly agrees with the live director at `WAVE1_ACTIVE @75T`, shows four authoritative enemy hitboxes/actors and all 13 arena anchors, and reports the selected seed/hash/performance counters.
- The ticket passes for both exact normal Wave 1–3 integration and the live Bell Proctor composite, including boss health/state/hazards/threat/hash evidence.
