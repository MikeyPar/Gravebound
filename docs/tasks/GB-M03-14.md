**Status:** Repository implementation complete; owner/platform and hosted rehearsal evidence pending.

## Objective

Define a secure, recoverable, cost-bounded private-test hosting architecture and an explicit Steamworks owner checklist without adding a Steam runtime dependency or claiming owner-controlled legal actions are complete.

1. `Gravebound_Production_GDD_v1_Canonical.md`: the product is native Windows first, keeps the Core prototype private, and reaches Steam Playtest only at a later public-playtest stage.

3. `Gravebound_Development_Roadmap_v1.md`: `GB-M03-14` requires Steamworks partner/legal/platform setup and a hosting/IaC ADR with no Steam runtime dependency.
