## Approved implementation contract

- Starter initialization is idempotent per character and atomically grants Worn Pine Crossbow, Worn Cracked Mark Lens, empty Armor/Charm, and two distinct Grant-provenance Red Tonic units in Belt index `0`.
- Reward and starter item UIDs are opaque 16-byte, domain-separated BLAKE3 derivations. Zero and collisions fail closed.
- Every consumable unit keeps its own UID and provenance; stack projections order and consume units by ascending unsigned UID.
- Reward equipment and Tonics enter `RunBackpack`; Tonics merge matching stacks before the lowest empty slot. Full legal capacity falls back to recipient-only `PersonalGround` for 60 seconds.
- Personal-ground expiry atomically records `Destroyed(ground_expired)` and its immutable ledger event.
- Reward request retry returns the stored plan/result without consuming RNG or mutating inventory again. Canonical request mismatch is an idempotency conflict.
- Wipeable development instances use only `core-dev.blake3.<full-manifest-hash>` content revisions.
